## 🛠 주요 기능

### 🍴 메뉴 선택
- 기본 선택 / 테마별 AI 추천
- 장바구니에 메뉴 담기

### 📍 위치 기반 매물 탐색
- 사용자 위치 권한 요청
- 반경 내 매물 목록 및 마커 표시

### 🛒 장바구니 관리
- 담은 메뉴 확인, 수량 조절, 삭제
- 총액 및 할인 계산

### 🚶 픽업 예약
- 예약 모달 → 예약 완료
- 픽업 코드 발급

### 🗺 경로 안내
- 픽업 코드 기반 예약 조회
- 픽업 최적 경로 표시

### 💡 알뜰팁 & 추천
- 반찬 활용법 제공
- AI 기반 추천 레시피 제공
